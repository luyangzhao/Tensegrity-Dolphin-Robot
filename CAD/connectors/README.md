# Connectors – Tail–Head Interface

This directory contains CAD files for the **connector components** that interface the tensegrity tail skeleton with the rigid head module.

## Files

- `Tail_Connector.stl`  
  Connector component used to mechanically interface the tail skeleton with the head module.  
  This part provides alignment and load transfer between the rigid head and the flexible tail.

## Notes

- The connector is compatible with all tail skeleton variants.
